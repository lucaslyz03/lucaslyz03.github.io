wsl-ubuntu20.04 配置

发表于 2020-11-13 更新于 2021-02-01

wsl2优化

自从wsl2更新以来，wsl的功能也日趋强大，随后我抱着尝试的心态使用了ubnutu wsl，发现十分好用，随即弃用虚拟机。下面记录对wsl优化过程：

<! more —>

初始配置

安装wsl ubuntu20.04

修改源

备份

sudo cp -v /etc/apt/sources.list /etc/apt/sources.list.backup

修改文件权限

sudo chmod 777 /etc/apt/sources.list

开始修改

vim /etc/apt/sources.list
  • 替换为（阿里源）
deb http://mirrors.aliyun.com/ubuntu/ focal main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ focal main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ focal-security main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ focal-security main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ focal-updates main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ focal-updates main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ focal-proposed main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ focal-proposed main restricted universe multiverse
deb http://mirrors.aliyun.com/ubuntu/ focal-backports main restricted universe multiverse
deb-src http://mirrors.aliyun.com/ubuntu/ focal-backports main restricted universe multiverse

保存退出

update

sudo apt update
sudo apt upgrade

E:\LinuxWSL\Ubuntu-18.04 是预先创建好的目录

LxRunOffline.exe move -n Ubuntu-18.04 -d >E:\LinuxWSL\Ubuntu18.04

遵循以下的格式 <发行版> 以 wsl.exe -l -v 命令执行结果中 NAME 字段的名称为准

LxRunOffline.exe move -n <发行版> -d <目的文件夹绝对路径>

zsh美化

安装zsh

cat /etc/shells

如果没有 zsh 需要安装

sudo apt-get install zsh

oh-my-zsh

sh -c "$(curl -fsSL https://raw.githubusercontent.com/robbyrussell/oh-my-zsh/master/tools/install.sh)"
  • 国内镜像替代

下载码云安装包

wget https://gitee.com/mirrors/oh-my-zsh/raw/master/tools/install.sh

编辑install.sh

找到以下部分

# Default settings
ZSH=${ZSH:-~/.oh-my-zsh}
REPO=${REPO:-ohmyzsh/ohmyzsh}
REMOTE=${REMOTE:-https://github.com/${REPO}.git}
BRANCH=${BRANCH:-master}

把

REPO=${REPO:-ohmyzsh/ohmyzsh}
REMOTE=${REMOTE:-https://github.com/${REPO}.git}

替换为

REPO=${REPO:-mirrors/oh-my-zsh}
REMOTE=${REMOTE:-https://gitee.com/${REPO}.git}

编辑后保存

修改仓库地址(备用)

# cd ~/.oh-my-zsh
# git remote set-url origin https://gitee.com/mirrors/oh-my-zsh.git
# git pull
sudo chmod +777 install.sh
./install.sh
vim ~/.zshrc
source ~/.zshrc
